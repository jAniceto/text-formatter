# text-formatter

This page contains a series of text formatting tools useful for programming.


### Features

Convert any text to:

- lowercase
- UPPERCASE
- Sentence case
- Capitalize Each Word
- lowercase-dashed-text (useful in Github anchors)


### Built with
- [Skeleton](http://getskeleton.com) CSS framework 
- Pure Javascript
